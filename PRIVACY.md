# Privacy Policy

This plugin sends prompts and messages to the Yandex Cloud Foundation Models API using your own API key and folder ID. No data is collected, stored, or shared by the plugin author.

All API requests are made directly from your Dify instance to Yandex Cloud servers (`llm.api.cloud.yandex.net`). Refer to [Yandex Cloud Privacy Policy](https://yandex.com/legal/confidential/) for how Yandex handles your data.
